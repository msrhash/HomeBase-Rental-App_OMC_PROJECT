# HomeBase Rental Management System

## Overview
HomeBase is a rental management platform designed to improve communication between property renters and agents, while simplifying the house viewing scheduling process. The system provides a centralized platform for managing property listings, bookings, and user interactions efficiently.

This group project was developed as part of the **Oh My Code (OMC)** project.

---

## Key Features

### User Side
- Browse available rental properties
- View property details
- Schedule house viewing appointments
- Communicate with property agents

### Admin Side
- Manage property listings (add/edit/remove)
- View and manage booking schedules
- Maintain system data
